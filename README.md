# MyWeatherData
Lokale Wetterapp zur Visualisierung von historischen Wetterdaten unter Nutzung der Daten aus dem Open Data Bereich des Climate Data Center des Deutschen Wetterdienst (DWD)

## Ziel: Darstellen und exportieren von historischen Wetterdaten über einen definierbaren Zeitraum
Zeitraum: innerhalb von 01.01.2015 bis 31.12.2025
Ort: nächst gelegener Wetterstation einer beliebigen Koordinate innerhalb von Deutschland 
Wetterdaten: Lufttemperatur, Niederschlag, Wind, Sonneneinstrahlung

## Umfang
- API zum Datenimport/-export vom Climate Data Center des DWD  
- Speichern und verwalten der Wetterdaten in einer lokalen Datenbank  
- User Interface zur Konfiguration von Ort, Zeitraum und darzustellenden Daten  
- Visuelle Darstellung der Wetterdaten entsprechend der Benutzervorgaben
